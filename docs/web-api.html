Slack Web API client

Slack Morphism provides you with a Web API methods client, that supports:

  • Make asynchronous Web API calls using provided tokens
  • Pagination and scrolling results of Slack API methods
  • Rate control, throttling and retrying API calls
  • Incoming Webhooks support
